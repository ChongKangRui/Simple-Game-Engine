Detailed Description

template<typename... Args>
class GameEngine::Signal< Args >

This documentation will briefing the usage of GameObject. GameObject is a class that can use to create a game object. A signal object may call multiple slots with the same signature.You can connect functions to the signal which will be called when the emit() method on the signal object is invoked. Any argument passed to emit() will be passed to the given functions.
